## Project Goals

This project focuses on:

- vanilla-first RedM infrastructure
- framework-agnostic architecture
- reproducible server setup
- modular resource organization
- minimal dependencies

---

## Philosophy

The repository is intentionally kept lightweight and infrastructure-oriented.

The goal is to provide a clean baseline for RedM servers without forcing heavy RP frameworks or large monolithic resource packs.
