# Troubleshooting

## FXServer Artifacts Missing

Ensure the `server/` directory contains extracted FXServer artifacts.

---

## Server Does Not Start

Check:

- server.cfg exists
- permissions.cfg exists
- ports 30120 TCP/UDP are open

---

## txAdmin Does Not Initialize

Ensure the latest FXServer artifacts are installed.

---

## Cannot Connect to Server

Check:

- firewall rules
- port forwarding
- public IP configuration
- server console logs
