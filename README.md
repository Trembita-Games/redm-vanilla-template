# redm-vanilla-template

Minimal vanilla-first RedM server template focused on clean infrastructure, reproducible setup and framework-agnostic architecture.

Created and maintained by members of the Oxbay community.

---

## Features

- Vanilla RedM baseline
- Clean repository structure
- txAdmin support
- Windows & Linux support
- Framework-agnostic setup
- Minimal dependencies
- Open-source friendly structure

---

## Philosophy

This project aims to provide a clean and minimal baseline for RedM servers without forcing heavy RP frameworks or large resource packs.

The goal is to keep the setup:

- vanilla-first
- modular
- reproducible
- infrastructure-oriented
- framework-agnostic

---

## Repository Structure

```txt
docs/       -> Documentation and guides
scripts/    -> Setup and maintenance scripts
resources/  -> Server resources and modules
```
